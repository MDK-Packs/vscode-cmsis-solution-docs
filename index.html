CMSIS Solution extension

The CMSIS Solution extension available for VS Code allows to create projects and build embedded applications that use CMSIS software packs. It is a user interface for the CMSIS-Toolbox that supports various compilation tools including Arm Compiler 6, GCC, IAR, and LLVM.
